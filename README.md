## 引言

利用`zeabur.app`做的防红链接，禁止生成非法网站，违反后果自行承担。
强烈推荐用Github和cloudflare的pages功能和以下托管平台自建防红 
Github防红html下载地址(open是直链，jump是跳转) 
https://github.com/ourls/ourls.github.io 
白名单自动通过域名: 
github.io，pages.dev，zeabur.app，netlify.app
